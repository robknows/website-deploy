Final year project

Adaptive compression for graph processing. During scans of a column, the algorithm I built identifies contiguous regions of the column and performs logical compressions. Using the compression information gathered during scans, future queries run faster by reducing the number of values that must be touched during a scan. I built a number of variations using different compression granularities and storage for the gathered column meta-information. I evaluated these algorithms using graph processing algorithms (pagerank, bfs) which perform many queries. The algorithms are just as applicable to relational query optimization as they are too graph query optimization. Key areas: query optimization, database auto-tuning, graph processing, compression
